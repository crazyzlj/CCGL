)

Constructor using IP address and port number.

Get a client for a MongoDB instance using IP address and port number.

Parameters
[in]hostIP address, e.g., 127.0.0.1
[in]portPort number, the default is 27017
Note
The constructor should not be used directly, since it does not check the validation of IP address and the status of the database, and also without mongoc_init(). So, please use the alternative initialization usage, MongoClient *client = MongoClient::Init(host, port).
